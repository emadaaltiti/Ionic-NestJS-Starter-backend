# server 


A Demo Server Side Nest js 

## Installation

```bash
$ cd Ionic-NestJS-Starter-backend
$ npm install 
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
navigate to http://localhost:3000 and you should see the template being served and rendered locally!.

```sh
http://localhost:3000
```
To Fetch items
